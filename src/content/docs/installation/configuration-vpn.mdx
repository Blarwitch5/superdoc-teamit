## Prérequis

- Windows 10 ou 11
- Identifiants VPN fournis par l'administrateur (serveur, nom d'utilisateur, mot de passe, éventuelle clé ou certificat)

## Configuration du VPN intégré Windows (IKEv2 / L2TP)

1. Aller dans **Paramètres → Réseau et Internet → VPN**.
2. Cliquer sur **Ajouter une connexion VPN**.
3. Renseigner :
   - **Fournisseur VPN** : Windows (intégré)
   - **Nom de la connexion** : ex. *VPN Team IT*
   - **Nom ou adresse du serveur** : adresse fournie par l'administrateur
   - **Type de réseau privé virtuel** : IKEv2 (ou L2TP/IPsec selon le serveur)
   - **Type d'informations d'identification** : Nom d'utilisateur et mot de passe
   - **Nom d'utilisateur / Mot de passe** : identifiants fournis
4. Cliquer sur **Enregistrer**.

## Connexion

1. Cliquer sur l'icône réseau dans la barre des tâches.
2. Sélectionner la connexion VPN créée.
3. Cliquer sur **Connexion**.

## Configuration OpenVPN (si applicable)

1. Télécharger **OpenVPN Connect** sur [openvpn.net](https://openvpn.net/client/).
2. Installer l'application.
3. Importer le fichier `.ovpn` fourni par l'administrateur : **Fichier → Importer profil → Depuis fichier**.
4. Saisir les identifiants et se connecter.

## Vérification

Une fois connecté, vérifier l'accès aux ressources internes (serveur de fichiers, intranet) via leur adresse IP privée.

## Dépannage

- **Connexion refusée** : vérifier que les identifiants sont corrects et que le service VPN du serveur est actif.
- **Timeout** : vérifier que le pare-feu réseau autorise les ports VPN (UDP 500, 4500 pour IKEv2 ; UDP 1194 pour OpenVPN).
- **Déconnexions fréquentes** : activer le **Reconnect automatique** dans les paramètres du client.
