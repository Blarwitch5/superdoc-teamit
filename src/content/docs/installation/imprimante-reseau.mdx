## Prérequis

- Imprimante connectée au réseau local (IP fixe recommandée)
- Adresse IP de l'imprimante (vérifiable sur l'écran de l'imprimante ou via le panneau de configuration réseau)
- Pilotes disponibles sur le site du fabricant

## Méthode 1 — Détection automatique

1. Aller dans **Paramètres → Bluetooth et appareils → Imprimantes et scanners**.
2. Cliquer sur **Ajouter un appareil**.
3. Windows recherche les imprimantes disponibles sur le réseau.
4. Sélectionner l'imprimante détectée et cliquer sur **Ajouter**.

Si l'imprimante n'apparaît pas, passer à la méthode 2.

## Méthode 2 — Ajout manuel par adresse IP

1. Aller dans **Paramètres → Bluetooth et appareils → Imprimantes et scanners**.
2. Cliquer sur **Ajouter manuellement** (lien en bas de la liste).
3. Choisir **Ajouter une imprimante à l'aide d'une adresse TCP/IP ou d'un nom d'hôte**.
4. Saisir l'adresse IP de l'imprimante.
5. Windows détecte le pilote ou propose d'en installer un.
6. Si le pilote n'est pas trouvé automatiquement : cliquer sur **Disque fourni** et pointer vers le pilote téléchargé.

## Définir comme imprimante par défaut

1. Aller dans **Paramètres → Bluetooth et appareils → Imprimantes et scanners**.
2. Cliquer sur l'imprimante → **Définir par défaut**.

## Vérification

Imprimer une page de test : **Paramètres de l'imprimante → Page de test**.

## Dépannage

- **Imprimante hors ligne** : vérifier la connexion réseau de l'imprimante et redémarrer le service `Spouleur d'impression` (services.msc).
- **Travaux bloqués** : ouvrir la file d'impression → annuler tous les travaux → relancer le spouleur.
- **Mauvais pilote** : désinstaller l'imprimante et réinstaller avec le pilote officiel du fabricant.
