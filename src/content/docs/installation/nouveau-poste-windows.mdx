## Étapes de configuration initiale

### 1. Windows Update

1. Allez dans **Paramètres** → **Windows Update** → **Rechercher les mises à jour**.
2. Installez toutes les mises à jour disponibles et redémarrez si demandé.
3. Répétez jusqu'à ce qu'aucune mise à jour ne soit disponible.

### 2. Rejoindre le domaine (si applicable)

1. **Paramètres** → **Système** → **À propos** → **Rejoindre un domaine**.
2. Entrez le nom de domaine de l'entreprise et les identifiants administrateur AD.
3. Redémarrez le poste.

### 3. Logiciels essentiels à installer

| Logiciel | Usage |
|---|---|
| Google Chrome | Navigateur principal |
| Microsoft Office | Suite bureautique |
| Adobe Acrobat Reader | Lecture PDF |
| 7-Zip | Compression de fichiers |
| FileZilla | Client FTP (si nécessaire) |
| Client VPN | Accès réseau distant (voir fiche *Configuration du VPN*) |

Téléchargez les logiciels depuis les sources officielles uniquement.

### 4. Configurer la boîte mail

Suivez la fiche *Configuration sur Outlook* pour ajouter la boîte mail du collaborateur.

### 5. Imprimante réseau

Suivez la fiche *Installation d'une imprimante réseau*.

### 6. Chiffrement du disque (BitLocker)

1. Recherchez **BitLocker** dans le menu Démarrer → **Gérer BitLocker**.
2. Activez BitLocker sur le lecteur C: et enregistrez la clé de récupération dans le dossier partagé IT.

## Checklist finale

- [ ] Windows à jour
- [ ] Compte domaine configuré
- [ ] Logiciels essentiels installés
- [ ] Boîte mail configurée dans Outlook
- [ ] Imprimante réseau accessible
- [ ] BitLocker activé, clé enregistrée
- [ ] VPN fonctionnel
