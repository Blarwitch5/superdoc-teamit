## Installation du système

- [ ] Installer Windows 11 (ou 10 selon la politique interne) avec une clé de licence valide
- [ ] Effectuer toutes les mises à jour Windows Update jusqu'à ce qu'il n'y en ait plus
- [ ] Vérifier et installer les pilotes manquants (carte graphique, réseau, son) via le gestionnaire de périphériques

## Sécurité de base

- [ ] Activer et configurer **Windows Defender** ou l'antivirus d'entreprise
- [ ] Activer le **pare-feu Windows**
- [ ] Activer **BitLocker** (chiffrement du disque) si la politique de sécurité l'exige
- [ ] Configurer les **mises à jour automatiques** sur Windows Update

## Comptes et accès

- [ ] Créer le compte utilisateur (local ou domaine) avec un mot de passe fort
- [ ] Désactiver le compte Administrateur local par défaut si non nécessaire
- [ ] Joindre le domaine Active Directory si applicable
- [ ] Configurer l'accès VPN (voir [Configuration VPN](/installation/configuration-vpn/))

## Logiciels essentiels

- [ ] Navigateur : Google Chrome ou Microsoft Edge (à jour)
- [ ] Suite bureautique : Microsoft 365 ou LibreOffice
- [ ] Client mail : Microsoft Outlook (voir [Configuration Outlook](/email/configuration-outlook/))
- [ ] PDF : Adobe Acrobat Reader
- [ ] 7-Zip ou WinRAR
- [ ] Client FTP : FileZilla (si nécessaire)

## Périphériques

- [ ] Configurer l'imprimante réseau (voir [Imprimante réseau](/installation/imprimante-reseau/))
- [ ] Vérifier le son, la caméra et le microphone (réunions en ligne)
- [ ] Configurer les écrans supplémentaires si applicable

## Messagerie et communication

- [ ] Configurer Outlook avec le compte e-mail professionnel
- [ ] Installer et configurer Microsoft Teams ou l'outil de communication interne

## Sauvegarde

- [ ] Vérifier que les dossiers utilisateur sont synchronisés (OneDrive, serveur de fichiers ou solution de sauvegarde)

## Vérification finale

- [ ] Redémarrer le poste et vérifier que tout fonctionne correctement
- [ ] Remettre les identifiants à l'utilisateur et lui demander de changer son mot de passe
- [ ] Faire signer le bon de réception du matériel
