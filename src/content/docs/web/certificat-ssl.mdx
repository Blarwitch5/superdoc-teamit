## Contexte

Un certificat SSL est indispensable pour sécuriser les échanges (HTTPS) et améliorer le référencement. Infomaniak propose des certificats **Let's Encrypt** gratuits et des certificats payants (DV, OV, EV).

## Activation d'un certificat Let's Encrypt (gratuit)

1. Se connecter à [manager.infomaniak.com](https://manager.infomaniak.com).
2. Aller dans **Hébergement Web → [nom du service] → SSL**.
3. Cliquer sur **Ajouter un certificat SSL**.
4. Choisir **Let's Encrypt (gratuit)**.
5. Sélectionner le ou les domaines à sécuriser.
6. Cliquer sur **Valider** — le certificat est émis en quelques minutes.

## Vérification

Une fois activé, vérifier que le site est accessible en HTTPS :

```
https://www.votredomaine.ch
```

L'icône cadenas doit être visible dans la barre d'adresse du navigateur.

## Renouvellement automatique

Les certificats Let's Encrypt sont renouvelés automatiquement par Infomaniak **30 jours avant expiration**. Aucune action manuelle n'est requise si le certificat a été créé via le Manager.

En cas d'échec de renouvellement automatique, Infomaniak envoie un e-mail d'alerte — vérifier les adresses de notification dans le compte.

## Redirection HTTP → HTTPS

Après l'activation du SSL, forcer la redirection HTTP vers HTTPS.

### Via le `.htaccess` (Apache)

Ajouter en début de fichier `.htaccess` à la racine du site :

```apache
RewriteEngine On
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]
```

### Via WordPress

Installer le plugin **Really Simple SSL** ou modifier l'URL du site dans **Réglages → Général**.

## Certificat payant (DV / OV / EV)

Pour les sites nécessitant un niveau de confiance élevé (e-commerce, banque) :

1. Aller dans **SSL → Acheter un certificat**.
2. Choisir le type (DV = validation de domaine, OV = organisation, EV = extended).
3. Suivre le processus de validation selon le type choisi.

Les certificats OV et EV nécessitent une validation de l'organisation (documents à fournir à l'autorité de certification).
