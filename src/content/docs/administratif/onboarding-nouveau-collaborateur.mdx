---

## Avant l'arrivée (J-2)

- [ ] Préparer le poste de travail (voir fiche *Mise en place d'un poste Windows*)
- [ ] Créer le compte Active Directory / compte entreprise
- [ ] Créer l'adresse e-mail (voir fiche *Création d'une adresse e-mail*)
- [ ] Créer les accès aux applications métier nécessaires
- [ ] Commander le matériel manquant si nécessaire (écran, clavier, souris, casque)

## Le jour de l'arrivée

### Remise du matériel

- [ ] Ordinateur portable ou fixe configuré
- [ ] Câbles et adaptateurs nécessaires
- [ ] Éventuellement : téléphone mobile professionnel

### Configuration sur place

- [ ] Connexion avec le compte utilisateur (Windows / AD)
- [ ] Configuration de la boîte mail sur Outlook (voir fiche *Configuration sur Outlook*)
- [ ] Configuration de la boîte mail sur iPhone si téléphone fourni (voir fiche *Configuration sur iPhone / iPad*)
- [ ] Installation du client VPN et test de connexion (voir fiche *Configuration du VPN*)
- [ ] Accès au serveur de fichiers partagés vérifié
- [ ] Imprimante configurée (voir fiche *Installation d'une imprimante réseau*)

### Accès aux services

- [ ] Invitation sur le compte Infomaniak si nécessaire (voir fiche *Création d'un accès Infomaniak*)
- [ ] Accès aux outils métier (CRM, ERP, intranet…) créés et testés

## Informations à transmettre au collaborateur

Remettez un document récapitulatif contenant :

- Adresse e-mail professionnelle
- Identifiant Windows / réseau
- Identifiants VPN
- Lien vers cette documentation IT

## Notes

- Le mot de passe initial doit être changé par le collaborateur dès la première connexion.
- Notez la date de remise du matériel et le numéro de série dans le registre IT.
