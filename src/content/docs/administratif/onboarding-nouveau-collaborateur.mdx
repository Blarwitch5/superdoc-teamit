## À préparer avant l'arrivée

### Matériel

- [ ] Commander ou préparer l'ordinateur (voir [Nouveau poste Windows](/installation/nouveau-poste-windows/))
- [ ] Préparer téléphone mobile si fourni
- [ ] Préparer badge et clés d'accès

### Comptes et accès

- [ ] Créer le compte Active Directory / utilisateur local Windows
- [ ] Créer l'adresse e-mail professionnelle (voir [Création d'une adresse e-mail](/email/creation-adresse-mail/))
- [ ] Ajouter aux boîtes mail partagées concernées
- [ ] Créer les accès aux outils métier (CRM, ERP, gestion de projet…)
- [ ] Ajouter à Infomaniak Manager si nécessaire (voir [Création d'accès Infomaniak](/administratif/creation-acces-infomaniak/))
- [ ] Configurer l'accès VPN si applicable (voir [Configuration VPN](/installation/configuration-vpn/))

## Le jour J

### Poste de travail

- [ ] Remettre le matériel et faire signer le bon de réception
- [ ] Connecter et vérifier le poste sur le réseau
- [ ] Configurer la messagerie Outlook (voir [Configuration Outlook](/email/configuration-outlook/))
- [ ] Configurer la messagerie sur mobile si applicable (voir [Configuration iPhone](/email/configuration-iphone/))
- [ ] Ajouter l'imprimante réseau (voir [Imprimante réseau](/installation/imprimante-reseau/))

### Accès et mots de passe

- [ ] Remettre les identifiants en main propre ou via canal sécurisé
- [ ] Demander le changement de mot de passe à la première connexion
- [ ] Activer l'authentification à deux facteurs (2FA) sur les services critiques

## Formation initiale

- [ ] Présenter les outils internes et leur usage
- [ ] Expliquer la politique de sauvegarde et de sécurité
- [ ] Présenter la documentation (cette plateforme)
- [ ] Désigner un référent technique pour les premières semaines

## Suivi

- [ ] Point à J+7 pour identifier les éventuels blocages techniques
- [ ] Point à J+30 pour valider que tous les accès sont opérationnels
