## Prérequis

- iPhone ou iPad avec iOS 14 ou version ultérieure
- Adresse e-mail et mot de passe Infomaniak

## Paramètres du serveur

| Paramètre | Valeur |
|-----------|--------|
| Serveur IMAP | `mail.infomaniak.com` |
| Port IMAP | `993` (SSL) |
| Serveur SMTP | `mail.infomaniak.com` |
| Port SMTP | `465` (SSL) |

## Procédure

1. Aller dans **Réglages → Mail → Comptes → Ajouter un compte**.
2. Choisir **Autre** (en bas de la liste).
3. Taper sur **Ajouter un compte Mail**.
4. Renseigner :
   - **Nom** : prénom et nom affiché à l'envoi
   - **Adresse** : adresse e-mail complète
   - **Mot de passe** : mot de passe du compte mail
   - **Description** : libellé affiché dans Mail (ex. : *Pro*)
5. Taper **Suivant** — iOS tente une configuration automatique. Si elle échoue, passer en manuel.
6. Sélectionner l'onglet **IMAP** et renseigner :
   - **Nom d'hôte** (entrant) : `mail.infomaniak.com`
   - **Nom d'utilisateur** : adresse e-mail complète
   - **Nom d'hôte** (sortant) : `mail.infomaniak.com`
7. Taper **Enregistrer**.

iOS configure automatiquement les ports SSL (993/465) lors de la validation.

## Synchronisation

Après l'ajout, aller dans **Réglages → Mail → Comptes → [votre compte]** et activer **Mail**, **Notes** si nécessaire.

## Dépannage courant

- **Impossible de vérifier l'identité du serveur** : accepter le certificat lors de la première connexion.
- **Mot de passe refusé en boucle** : supprimer le compte et le recréer ; vérifier l'absence d'espaces dans le mot de passe.
- **Emails non reçus** : vérifier que le dossier de réception est bien mappé sur `INBOX`.
