## Prérequis

- Microsoft Outlook installé (2016 ou version ultérieure)
- Adresse e-mail et mot de passe Infomaniak

## Paramètres du serveur

| Paramètre | Valeur |
|-----------|--------|
| Serveur entrant (IMAP) | `mail.infomaniak.com` |
| Port IMAP | `993` (SSL/TLS) |
| Serveur sortant (SMTP) | `mail.infomaniak.com` |
| Port SMTP | `465` (SSL/TLS) ou `587` (STARTTLS) |
| Authentification | Nom d'utilisateur + mot de passe |

## Procédure

1. Ouvrir Outlook et aller dans **Fichier → Ajouter un compte**.
2. Saisir l'adresse e-mail complète, puis cliquer sur **Options avancées → Configurer mon compte manuellement**.
3. Choisir **IMAP**.
4. Renseigner les paramètres serveur entrant :
   - Serveur : `mail.infomaniak.com`
   - Port : `993`
   - Chiffrement : **SSL/TLS**
5. Renseigner les paramètres serveur sortant :
   - Serveur : `mail.infomaniak.com`
   - Port : `465`
   - Chiffrement : **SSL/TLS**
6. Saisir le nom d'utilisateur (adresse e-mail complète) et le mot de passe.
7. Cliquer sur **Se connecter**.

## Vérification

Envoyer un e-mail test depuis le compte pour valider la configuration. Si l'envoi échoue, vérifier que le port 465 n'est pas bloqué par un pare-feu ou antivirus — essayer le port 587 avec STARTTLS en alternative.

## Dépannage courant

- **Mot de passe refusé** : vérifier que le mot de passe est bien celui du compte Infomaniak Mail (distinct du mot de passe manager).
- **Certificat non reconnu** : accepter le certificat ou mettre à jour Outlook.
- **Dossiers non synchronisés** : aller dans **Paramètres du compte → Paramètres des dossiers** et forcer la synchronisation.
