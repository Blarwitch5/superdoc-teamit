## Accès

1. Se connecter à [manager.infomaniak.com](https://manager.infomaniak.com).
2. Dans le menu latéral, aller dans **Hébergement Web → [nom du service] → E-mail**.

## Création de l'adresse

1. Cliquer sur **Créer une adresse e-mail**.
2. Renseigner :
   - **Adresse** : partie avant le `@` (ex. : `prenom.nom`)
   - **Domaine** : sélectionner le domaine dans la liste déroulante
   - **Mot de passe** : générer ou saisir un mot de passe fort (min. 12 caractères, majuscule, chiffre, symbole)
   - **Quota** : laisser la valeur par défaut ou ajuster selon les besoins
3. Cliquer sur **Créer**.

## Après la création

- **Communiquer les identifiants** à l'utilisateur via un canal sécurisé (jamais par e-mail non chiffré).
- **Tester l'accès** via [mail.infomaniak.com](https://mail.infomaniak.com) (webmail).
- **Configurer le client mail** selon les fiches [Configuration Outlook](/email/configuration-outlook/) ou [Configuration iPhone](/email/configuration-iphone/).

## Options supplémentaires

| Option | Utilité |
|--------|---------|
| Alias | Ajouter une adresse supplémentaire sur la même boîte |
| Redirection | Transférer les e-mails vers une autre adresse |
| Répondeur automatique | Message d'absence |
| Filtre anti-spam | Activer / ajuster le niveau de filtrage |

## Notes

- Le quota par défaut est généralement de **5 Go** ; augmenter si l'utilisateur reçoit beaucoup de pièces jointes.
- Préférer le format `prenom.nom@domaine.ch` pour la cohérence.
